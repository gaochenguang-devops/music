# Sonora 参考图 UI 重设计

## 视觉方向

采用参考图的冷白桌面工作台语言：浅灰蓝应用背景、白色浮层工具区、低对比边框与阴影、青色主操作。界面保持轻盈但不牺牲播放器的信息密度。圆角限制在 14 px，重复列表项使用更小的 8 px 圆角。

## 信息层级

顶部浮栏显示小尺寸封面、标题、歌手与专辑；导入音乐是主要操作，歌词和输出设备为次级操作。中央白色舞台只承载歌词，当前双语歌词用浅青背景、青色边框和大号深色文字突出。右侧播放列表为独立工具面板，可收起并释放空间。底部控制坞上排为进度，下排采用左侧模式/倍速、中间固定播放控制、右侧音量的稳定三栏布局。

## 响应式与状态

顶部左右区域按窗口宽度分配；歌词宽度始终来自中央舞台实际剩余区域。列表收起状态继续持久化。封面缺失时使用带唱片圆环的中性占位，不依赖网络素材。悬停只使用轻微底色变化，避免密集动画影响操作。

## 验证

使用真实 ID3/LRC 曲目验证封面、双语歌词和列表；分别检查列表展开与收起状态；在 Windows 150% DPI 下截图，确认文字不裁切、控制组居中、面板间距一致。最后运行测试、严格 Clippy 和 release 构建。
